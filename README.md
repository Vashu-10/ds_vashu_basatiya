# ds_vashu_basatiya
# Trader Behavior vs Market Sentiment Analysis

This repository contains a data science analysis exploring the relationship
between Bitcoin market sentiment (Fear & Greed Index) and trader behavior
using historical trade data from Hyperliquid.

## Project Structure
- notebook_1.ipynb : Main analysis notebook (Google Colab)
- csv_files/merged_data.csv : Cleaned and merged dataset
- outputs/ : Visualizations generated during analysis
- ds_report.pdf : Final summarized insights and conclusions

## Execution
All analysis was performed in Google Colab.
